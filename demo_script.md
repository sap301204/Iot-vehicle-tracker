# Demo Script & Screenshot Plan

Demo steps:
1. Show hardware (ESP32, GPS, Relay) and wiring.
2. Open Serial Monitor to show WiFi and GPS fix messages.
3. Start Node-RED, show dashboard and live coordinate updates.
4. Open Blynk app to show mobile dashboard; press LOCK to immobilize (relay click).
5. Simulate movement via mosquitto_pub; observe alert on Blynk and Node-RED.
6. Show Grafana speed graph (sample data) and route history.
